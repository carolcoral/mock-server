# 版本变更说明

## v2.0.4 (2026-06-12)

> UI/UX 深度美化与配置增强。

### ✨ 新增
- **自定义链接 SVG 图标**：页脚自定义链接支持设置 SVG 图标，实时预览，无图标时回退默认样式
- **页脚可选配置**：版权、友情链接、博客、GitHub、邮箱、自定义链接均支持独立开关，全部关闭时自动隐藏页脚区域

### 🎨 UI 美化
- **登录页**：标题渐变流动动画 + 六边形盒子 SVG 图标，移除底部版权信息
- **侧边栏**：深色渐变背景 + Canvas 动态游走发光线条 + 顶部氛围光晕
- **侧边栏菜单**：hover 左侧光柱滑入 + 渐变高亮 + 图标缩放动效，选中态发光边框
- **侧边栏 Logo**：彩色渐变文字 + 流动动画 + 六边形图标

### 🐛 修复
- 页脚设置保存后页面不自动刷新同步
- `setup-env.sh` 默认执行安装，移除交互确认提示

### ⚡ 优化
- 侧边栏动态线条透明度调低，菜单文字更突出

## v2.0.3 (2026-06-11)

> 安全漏洞修复，`npm audit` 清零。

### 🔒 依赖升级
- **axios** `1.13.6` → `1.15.1`（3 个 CVE：HTTP 劫持、RCE、空字节编码）
- **postcss** `8.5.6` → `8.5.10`（XSS 漏洞）
- **vite** `5.x` → `6.x`，配套升级 vitest、@vitejs/plugin-vue
- **Spring Boot** `3.2.0` → `3.2.12`
- 间接依赖（lodash、picomatch 等）同步修复，`npm audit` 零漏洞

### 🔒 Docker 安全加固
- 基础镜像升级，非 root 用户运行
- 容器只读文件系统、禁止提权、最小权限
- npm 构建阶段禁止脚本执行

## v2.0.2 (2026-06-11)

> 新功能与核心问题修复。

### 🚀 新增
- **登录页 Bing 每日图片**：随机获取 Bing 每日推荐图片作为背景，2 秒超时自动回退默认图
- **Docker 完整部署方案**：多阶段构建 Dockerfile、docker-compose 编排、一键构建推送脚本

### 🐛 修复
- **Mock 随机返回**：修复条件匹配优先级导致的随机功能失效，权重和随机选择恢复正常
- **CORS / 静态资源 401**：修复生产环境下 Bing 图片跨域和静态资源（图片、字体、JS/CSS）认证拦截问题
- **Favicon**：修复不存在的 vite.svg 引用

## v2.0.0 (2026-06-10)

> 基于 v1.0.2 的重大版本升级。

### 🚀 新增
- **多语言国际化**：中文简体、English、日本語三语言支持，全站实时切换
- **自定义响应处理器**：Java/JavaScript 动态代码编译器，保存即生效，内置字段脱敏等插件
- **系统监控**：CPU、内存、磁盘实时监控，JVM 堆详情，颜色指示器
- **系统公告**：Markdown 公告编辑，优先级分级，首页展示
- **日期格式配置**：YYYY-MM-DD / DD/MM/YYYY / MM/DD/YYYY 可选，全局统一

### ✨ 增强
- **安全**：登录失败锁定、IP 白名单、JWT 过期可配
- **用户管理**：管理员完整编辑用户信息，权限分级
- **接口管理**：请求参数管理（PATH/QUERY/BODY/FILE），代码处理器集成
- **系统配置**：Axios 超时、配置持久化、统一配置接口

### 🐛 修复
- 语言切换不全站生效、i18n key 显示异常、日期格式不持久化
- 日语环境按钮错位、创建时间换行、401 页面刷新拦截
- 管理员编辑用户无权限、修改密码不退出、个人信息刷新问题
