# 版本变更说明

## v2.0.0 (2026-06-10)

> 基于 v1.0.2 的重大版本升级，新增多语言国际化、自定义响应处理器、系统监控、用户管理增强等功能。

### 🚀 新增功能

#### 多语言国际化支持
- 新增 **中文简体 (zh-CN)**、**English (en-US)**、**日本語 (ja-JP)** 三种系统语言
- 系统设置中可切换语言，保存后全站实时生效（导航栏、表格、按钮、提示消息等）
- 个人信息页面支持设置用户语言偏好
- 表单验证规则响应式适配语言切换

#### 自定义响应处理器
- 新增 `DynamicCompiler` 动态编译器，支持在接口配置中编写 **Java/JavaScript** 代码
- 代码保存后**实时编译生效**，无需重启服务
- 内置 `TransformerRegistry` 插件注册中心
- 预置三个转换器插件：`FieldMaskTransformer`（字段脱敏）、`ResponseWrapperTransformer`（响应包装）、`TimestampAppenderTransformer`（时间戳追加）

#### 系统监控
- 系统设置新增**系统信息**页面，实时展示：
  - CPU 使用率（5秒自动刷新）
  - 内存使用率及 JVM 堆内存详情
  - 磁盘使用率及空间详情
  - 系统运行时间、Java 版本、数据库信息
  - 安全环境变量查看
- 使用率颜色指示：绿色(正常) → 橙色(>50%) → 红色(>80%)

#### 系统公告
- 管理员可创建/编辑/删除系统公告
- 公告内容支持 **Markdown** 格式
- 支持优先级设置（低/普通/高/紧急）
- 首页展示当前启用的公告

#### 日期格式配置
- 系统设置中可选择日期显示格式：`YYYY-MM-DD` / `DD/MM/YYYY` / `MM/DD/YYYY`
- 全局日期格式化工具 `dateFormat.js`，所有表格统一使用
- 配置持久化到后端，跨设备同步

#### 使用说明文档
- 首页新增**使用说明**按钮，打开 `USER_GUIDE.md` 文档
- 文档内容随版本更新同步维护

#### 页脚
- 页面底部新增页脚区域，居中展示版权信息 `© 2026 carolcoral`
- 彩色图标链接：友情链接、博客、GitHub、邮箱

### ✨ 功能增强

#### 用户管理
- 管理员可完整编辑用户信息（角色、邮箱、启用状态）
- 新增 `PUT /api/users/{userId}` 管理员更新用户端点
- 用户管理页面完全国际化

#### 项目管理
- 项目管理页面完全国际化，支持多语言
- 成员管理功能增强（OWNER/MEMBER/VIEWER 角色）
- 日期字段使用配置的日期格式显示

#### 接口管理
- 接口管理页面完全国际化（含响应管理、请求参数管理）
- 自定义代码处理器集成到接口详情页
- 新增请求参数管理（PATH/QUERY/BODY/FILE 类型）

#### 安全配置
- 新增登录失败锁定机制（可配置最大尝试次数和锁定时长）
- IP 白名单支持
- JWT Token 过期时间可配置
- `/profile` 路径添加 SPA 路由支持，修复刷新 401 问题

#### 系统配置
- Mock 配置增加 Axios 超时时间设置
- 配置持久化到数据库 `t_system_config` 表
- 新增 `SystemConfigController` 统一配置读写接口

### 🐛 问题修复
- 修复切换系统语言仅设置页面生效的问题
- 修复语言下拉框显示原始 i18n key 的问题
- 修复日语环境下用户管理页按钮错位
- 修复创建时间列日期换行问题
- 修复日期格式设置不持久化的问题
- 修复使用说明页面返回 401 的问题
- 修复 `/profile` 页面刷新返回 401 的问题
- 修复管理员编辑用户提示无权限的问题
- 修复个人信息修改语言后页面不刷新的问题
- 修复修改密码后不自动退出的问题
